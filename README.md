## ðŸš€ Getting Started

### 1. Prerequisites
- **Node.js** (v14 or higher)
- **Cloudinary Account**: A free account is sufficient. You will need your Cloud Name, API Key, and API Secret.

### 2. Environment Setup
Create a `.env` file in the root directory and add the following:

```env
PORT=3000
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
PUBLIC_BASE_URL=http://localhost:3000
```

### 3. Installation & Run
```bash
# Install dependencies
npm install

# Start the server
npm start
```

Open your browser and navigate to `http://localhost:3000`.

## ðŸ”§ Admin & Advanced

### Double Strip Printing (Admin)
If you are managing the booth and want to print the current session's double strip from a separate device (e.g., a phone connected to a printer):

1.  Ensure the session has reached the **Done** screen.
2.  Navigate to the following URL on your admin device:
    `https://res.cloudinary.com/<your-cloud-name>/image/upload/photobooth-print.png`
3.  This URL always points to the **latest** double strip generated by the booth.
4.  Print directly from your browser or save and use a printing app (like Canon SELPHY).

> [!NOTE]
> The `photobooth-print` ID is overwritten with every new session, so you only ever see the most recent capture.

## ðŸ›  Troubleshooting

- **Camera not starting?** Ensure you've granted camera permissions in your browser.
- **QR codes not loading?** Check that your Cloudinary credentials in `.env` are correct.
- **Phone can't reach the link?** Ensure your phone and the host computer are on the same Wi-Fi, and your `PUBLIC_BASE_URL` is set to your machine's local IP.
